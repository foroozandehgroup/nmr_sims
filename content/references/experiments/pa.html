spectrum(zf_factor=1.0, lb=None)[source]¶

Return the spectrum associated with a simulation.

Parameters

  • zf_factor (float) – The ratio between the number of points in the final spectrum, generated by zero-filling the FID, and the FID itself in each dimension. 1.0 (default) means no zero-filling is applied. Should be >= 1.0.

  • lb (Optional[float]) – Line-broadening factor for exponential window function. Default option (None), will apply an exponential window such that the final point will be shrunk to 1/1000 of its original value.

Return type

Tuple[ndarray, ndarray]

Returns

  • shifts – The chemical shifts sampled.

  • spectrum – The spectrum generated from the FID.

  • labels – Axis label for plotting purposes.
